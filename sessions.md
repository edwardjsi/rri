* **Session 1: Infrastructure & Backend (Day 1)**
    * Initialize AWS CDK project.
    * Provision DynamoDB table, API Gateway, and Lambda functions.
    * Set up GitHub Actions CI/CD pipeline for automated deployment.
* **Session 2: Frontend & Logic Integration (Day 2)**
    * Build Next.js UI components (Progress bar, Question cards).
    * Implement the 15-question logic and state management.
    * Connect frontend to the AWS API Gateway.
* **Session 3: Styling, Lead Capture & Polish (Day 3)**
    * Implement Tailwind CSS for a premium, trustworthy design.
    * Build the final Lead Capture form and Results rendering.
    * End-to-end testing and final production deployment.
