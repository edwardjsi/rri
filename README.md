# Sovereign Retirement Reality Check™ (RRI)

An AWS serverless web application designed to assess financial resilience across Protection, Income, Market, and Inflation layers.

## Tech Stack
* **Frontend:** Next.js (React), Tailwind CSS
* **Backend:** AWS Lambda, API Gateway
* **Database:** DynamoDB
* **DevOps:** AWS CDK, GitHub Actions

## Quick Start (Local Development)
1. Clone the repository.
2. Run `npm install` in the `/frontend` directory.
3. Run `npm run dev` to start the local Next.js server.
4. For infrastructure, navigate to `/infra` and use `cdk deploy` (requires AWS CLI configuration).
